Package not installed

abstractGikTask does not seem to be installed

Or it has been installed more recently than this documentation. In this case, you need to configure and install again this documentation.

To install abstractGikTask, you need either
  • to get the source code through git: git-clone git+ssh://[softs|git].laas.fr/git/jrl/abstractGikTask, configure, compile and install the package.
  • or install it through robotpkg: cd ${ROBOTPKG_BASE}/robotpkg/interfaces/jrl-interface-giktask; make update

Note that you will need to configure and install again this documentation to activate the link to abstractGikTask.
