slam3DPlanner does not seem to be installed

Or it has been installed more recently than this documentation. In this case, you need to configure and install again this documentation.

To install slam3DPlanner, you need to
  • get the source code through git: git-clone git+ssh://[softs|git].laas.fr/git/jrl/slam3DPlanner,
  • configure, compile and install the package.

Note that you will need to configure and install again this documentation to activate the link to slam3DPlanner.
